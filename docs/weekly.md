# 周报制度

一般情况下在每周五三点前需要通过邮件向上级发送周报，描述本周的工作内容。如有特殊情况，发送时间可以延后。

为了方便周报的汇总整理，小组成员需要统一周报格式，周报模板已提供，按需下载使用，在表格中填写完周报后，复制粘贴到邮件中即可。
