# 打造舒适的开发环境

# Chrome 插件

目前最流行，最强大的浏览器，也是前端开发必备的浏览器。下面我们来介绍使用Chrome浏览器的一些小技巧。

  • 沙拉查词 必装的插件，划词翻译功能是我目前用过的最舒服的；
  • fe-helper 百度推出的非常强大的前端工具，功能非常的强悍，如 JSON查看、代码美化、代码压缩、图片 Base64、二维码生成器等等；
  • vue-devtools 开发 Vue 应用时的辅助工具；
  • octo-tree 方便你查看 GitHub 项目结构
  • octo-linker 查看 GitHub 文件时可以一键打开引入文件的路径；

# VSCode 快捷键

# 左边栏快捷键，定制化

cmd + k + s -> keybinds.json

添加以下配置

{
  "key": "cmd+1",
  "command": "workbench.view.explorer"
},
{
  "key": "cmd+2",
  "command": "workbench.view.search"
},
{
  "key": "cmd+3",
  "command": "workbench.view.scm"
},
{
  "key": "cmd+4",
  "command": "workbench.view.debug"
},
{
  "key": "cmd+5",
  "command": "workbench.view.extensions"
},

# VSCode 插件

  • xiyun 可以快速生成符合禧云生态的标准页面。
  • JavaScript (ES6) code snippets 可以给文件(js、ts、vue、jsx、tsx) 增加统一标准的 Snippets。
  • Code Spell Checker 英文单词拼写错误检查插件；这个插件是从 vue-next 项目接触到的，可以有效避免拼写错误等低级错误。
  • Regex Previewer 正则表达式辅助工具，在写正则表达式时右侧能实时的看到匹配效果。
