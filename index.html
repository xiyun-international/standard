# 介绍

为了降低沟通成本，禧云需要建立前端规范，

# 为什么需要规范?

降低新成员融入团队的成本；
提高开发效率以及团队协作效率，降低沟通成本；
实现高度统一的编码风格，方便 Review 提高项目的维护性；
规范是实现自动化的基础，是一个团队知识沉淀的直接输出；

# 规范包含哪些内容?

前端规范并不单单指编码规范，涉及到前端开发工作的多个方面；
例如代码库的管理、前后端协作、代码规范、兼容性规范；
